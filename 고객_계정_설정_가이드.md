# 모바일 청첩장 영구 소장 — 계정 설정 가이드

안녕하세요! 영구 소장형 청첩장 제작을 신청해 주셔서 감사합니다.  
아래 순서대로 계정을 만들어 주시면, 나머지 작업은 제가 모두 진행해 드리겠습니다. 😊

---

## 준비물

- 스마트폰 또는 컴퓨터
- 약 10~15분

---

## STEP 1 — 구글 계정 만들기

> 이미 구글(Gmail) 계정이 있으시면 이 단계는 건너뛰셔도 됩니다.

1. [accounts.google.com/signup](https://accounts.google.com/signup) 접속
2. 성명, 이메일 주소, 비밀번호 입력 후 계정 생성

**권장 비밀번호 규칙:**  
영문 + 숫자 + 특수문자 조합 8자리 이상  
예) `Wedding2025!`

> ✅ 이메일 주소와 비밀번호를 메모해 두세요. 이후 모든 서비스에 동일하게 사용하실 수 있습니다.

---

## STEP 2 — GitHub 계정 만들기

GitHub는 청첩장 파일이 저장되는 곳입니다.

1. [github.com/signup](https://github.com/signup) 접속
2. **"Continue with Google"** 버튼 클릭 → 구글 계정으로 간편 가입
3. Username(사용자명) 설정  
   예) `jinho-naeun-wedding`
4. 가입 완료

> ✅ Username을 메모해 두세요.

---

## STEP 3 — Cloudflare 계정 만들기

Cloudflare는 청첩장이 실제로 인터넷에 올라가는 곳입니다.

1. [dash.cloudflare.com/sign-up](https://dash.cloudflare.com/sign-up) 접속
2. **"Sign up with Google"** 버튼 클릭 → 구글 계정으로 간편 가입
3. 가입 완료

---

## STEP 4 — 계정 정보 전달

아래 정보를 저에게 보내주세요.

```
구글 이메일: 
GitHub 사용자명(Username): 
```

> 비밀번호는 보내지 않으셔도 됩니다.  
> 작업은 공개된 파일 한 개를 올려드리는 방식으로 진행됩니다.

---

## STEP 5 — 작업 완료 후

작업이 완료되면 아래 두 가지 링크를 보내드립니다.

| 항목 | 내용 |
|---|---|
| 청첩장 주소 | `https://xxx.pages.dev` |
| 관리 페이지 | 참석 응답 · 방명록 확인용 |

**받으신 후 비밀번호를 변경하고 안전하게 보관해 주세요.**  
(구글 계정 비밀번호 변경 → GitHub, Cloudflare 모두 자동 반영)

---

## 자주 묻는 질문

**Q. 나중에 청첩장 내용을 수정할 수 있나요?**  
A. 사진이나 내용 수정이 필요하시면 말씀해 주세요. 추가 비용 없이 1회 수정해 드립니다.

**Q. 청첩장이 영구적으로 유지되나요?**  
A. 네, 계정이 유지되는 한 영구 보존됩니다. 구글·GitHub·Cloudflare 모두 무료 계정으로 유지 가능합니다.

**Q. 계정을 공유하는 게 불안한데요.**  
A. GitHub와 Cloudflare는 구글 계정으로 로그인하는 방식이라 별도 비밀번호가 없습니다. 작업 완료 후 구글 계정 비밀번호만 변경하시면 됩니다.
